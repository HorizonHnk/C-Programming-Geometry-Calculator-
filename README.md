## 🎯 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software required!

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/HorizonHnk/C-Programming-Geometry-Calculator.git
   ```

2. **Navigate to project directory**
   ```bash
   cd C-Programming-Geometry-Calculator
   ```

3. **Open in browser**
   ```bash
   # Simply open index.html in your preferred browser
   open index.html  # macOS
   start index.html # Windows
   xdg-open index.html # Linux
   ```

### Alternative: Direct Download
1. Click **"Code"** → **"Download ZIP"**
2. Extract the ZIP file
3. Double-click `index.html` to launch

## 🧮 Supported Calculations

<details>
<summary><strong>3D Shapes (Volume)</strong></summary>

| Shape | Formula | Variables |
|-------|---------|-----------|
| **Sphere** | `V = (4/3)πr³` | radius |
| **Cone** | `V = (1/3)πr²h` | radius, height |
| **Cube** | `V = s³` | side length |
| **Cylinder** | `V = πr²h` | radius, height |
| **Cuboid** | `V = l×w×h` | length, width, height |
| **Hemisphere** | `V = (2/3)πr³` | radius |

</details>

<details>
<summary><strong>2D Shapes (Area)</strong></summary>

| Shape | Formula | Variables |
|-------|---------|-----------|
| **Square** | `A = s²` | side length |
| **Rectangle** | `A = l×w` | length, width |
| **Triangle** | `A = (1/2)bh` | base, height |
| **Circle** | `A = πr²` | radius |
| **Ellipse** | `A = πab` | semi-major, semi-minor axis |
| **Trapezoid** | `A = (1/2)(a+b)h` | parallel sides, height |

</details>
